GoLeftAndRight
2012 03 14
A small racing game in which the player steer the car with only one button. Different zones on the track determines which way the car turns.
The name is half-stolen from a game that helped inspire this project: Go Left.

Tools used:
  • The game is built in haXe and is exclusive for the flash 9 platform and above.

  • To compile the library assets SWFMill was used.

  • No additional AS library or framework was used except for Troy Gilbert's collision detection which I ported to haXe first.

  • All graphic assets were made using Xara Xtreme and GIMP with a wacom bamboo MTE-450.

  • The music was composed on a machinedrum sps-1UW and a monomachine sfx-60 mkII and recorded with an olympus LS-11.

  • For sound effects a Thingamagoop 2 was involved too.

  • LAME was used for MP3 encoding. If you have trouble with gaps in MP3 files the tip of the day is the LAME nogap option!


Planned release: Q4 2011 Q1 2012. Currently up for sponsor viewing on Flash Game License.
